# 小娇使用方法页面

## 功能概述

"小娇使用方法"是一个直观、易用的功能使用指南页面，帮助用户快速了解智能运维后台管理系统的各项功能和操作流程。

## 页面特性

### 1. 功能分类
- **Spoke类别**：直接操作类功能（蓝紫色渐变）
  - 交付物管理
  - CMDB管理
  - 测试环境支持
  - 运维执行
  - 智能巡检
  - 事件支持

- **Hub类别**：汇总分析类功能（粉红色渐变）
  - 事件统计
  - 知识库
  - 合作方大屏
  - 智能总结

### 2. 核心功能

#### 搜索和筛选
- **全局搜索**：支持按功能名称、描述、关键词实时搜索
- **分类筛选**：按Spoke/Hub类别快速筛选
- **左侧导航**：分类树形菜单，支持快速跳转

#### 视图模式
- **网格视图**：卡片式布局，适合浏览和发现
- **列表视图**：紧凑列表，适合快速查找

#### 功能卡片
每个功能卡片包含：
- 功能图标和名称
- 类别标签（Spoke/Hub）
- 新功能标识
- 简要描述
- 快速查看详情入口

#### 详细信息
点击卡片查看完整使用指南：
- 功能类型和分类
- 详细描述
- 使用场景
- 操作步骤（步骤式展示）
- 注意事项
- 快捷跳转到功能页面

#### 快捷操作
- 右侧常用功能快速入口
- 热门功能推荐

## 访问方式

1. **主菜单访问**：在左侧导航栏底部点击"❓ 小娇使用方法"
2. **路由访问**：直接访问 `/help/user-guide`

## 扩展说明

### 添加新功能
在 `UserGuideView.vue` 的 `features` 数组中添加新功能项：

```typescript
{
  id: 20,
  name: '功能名称',
  type: 'spoke', // 或 'hub'
  category: '功能分类',
  icon: IconComponent, // Element Plus图标
  description: '功能简要描述',
  scenario: '使用场景说明',
  steps: ['步骤1', '步骤2', '步骤3'],
  tips: ['注意事项1', '注意事项2'],
  isNew: true, // 是否显示新功能标签
  route: '/path/to/feature' // 可选：功能页面路由
}
```

### 添加新分类
在 `spokeCategories` 或 `hubCategories` 中添加：

```typescript
{ id: 'category-id', name: '分类名称' }
```

## 设计特点

1. **渐变色彩**：Spoke使用蓝紫渐变，Hub使用粉红渐变，视觉区分明显
2. **卡片悬停效果**：鼠标悬停时卡片上浮，增强交互反馈
3. **响应式布局**：自适应网格布局，支持不同屏幕尺寸
4. **玻璃态设计**：与系统整体设计风格保持一致
5. **流畅动画**：过渡动画提升用户体验
